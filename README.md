# Prueba Técnica Sofka

Este proyecto corresponde a la prueba técnica solicitada por Sofka.
Incluye un backend en Spring Boot y un frontend en React.

Nombre: Diego Casa

# Tecnologías

# Backend
- Java 17
- Spring Boot
- JPA / Hibernate
- PostgreSQL
- JUnit + MockMvc

# Frontend
- React + Vite
- TypeScript
- Vitest
- Testing Library

# Ejecución del proyecto (recomendado)

El proyecto está dockerizado.  
Para levantar todo el sistema (backend, frontend y base de datos):

docker-compose up
